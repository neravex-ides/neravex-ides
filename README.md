# 🚀 Global Tech Services | SOFT-DISTRO

An ultra-modern, interactive software distribution platform built with **React**, **Vite**, and **Tailwind CSS**.

## ✨ Features
* 🌌 **3D Particle Background** - Immersive visual experience using Three.js.
* 🧲 **Magnetic Buttons** - High-end interactive UI components.
* 📱 **Fully Responsive** - Works perfectly on Mobile, Tablet, and Desktop.
* ⚡ **Performance Optimized** - Lightning-fast loading thanks to Vite 6.

## 🛠️ Tech Stack
* **Frontend:** React.js
* **Styling:** Tailwind CSS
* **Animations:** Framer Motion
* **Bundler:** Vite

## 🛠️ Installation
1. Clone the repo: `git clone https://github.com/neravex-ides/soft-distro.git`
2. Install dependencies: `npm install`
3. Run locally: `npm run dev`
